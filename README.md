# $p$-adic numbers in Macaulay2

This repository hosts the following:

* The [Padic](https://www.macaulay2.com/doc/Macaulay2/share/doc/Macaulay2/Padic/html/) package for Macaulay2.  This source should stay in sync with the [source in the Macaulay2 development branch](https://github.com/Macaulay2/M2/blob/development/M2/Macaulay2/packages/Padic.m2).
* The LaTeX source for the paper [Implementing p-adic numbers in Macaulay2 using its foreign function interface and FLINT](https://arxiv.org/abs/2604.16799).
