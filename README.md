# $p$-adic numbers in Macaulay2

This repository hosts the following:

* The [Padic](https://d-torrance.github.io/macaulay2-padic/) package for Macaulay2.  This source should stay in sync with the [source in the Macaulay2 development branch](https://github.com/Macaulay2/M2/blob/development/M2/Macaulay2/packages/Padic.m2).
* The LaTeX source for the paper [Implementing p-adic numbers in Macaulay2 using its foreign function interface and FLINT](https://arxiv.org/abs/2604.16799).
* The [slides](https://d-torrance.github.io/macaulay2-padic/icms) and [code demo](https://nbviewer.org/github/d-torrance/macaulay2-padic/blob/master/icms/macaulay2-padic.ipynb) from my talk at [ICMS 2026](https://icms-conference.org/2026/).
